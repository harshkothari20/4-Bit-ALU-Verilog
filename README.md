# 4-bit ALU Design using Verilog (Vivado)

## Overview
Designed and verified a 4-bit Arithmetic Logic Unit (ALU) in Verilog using Xilinx Vivado.

## Operations Implemented
- Addition
- Subtraction
- AND
- OR
- XOR
- NOT
- Shift Left
- Shift Right

## Features
- Carry flag
- Overflow flag
- Zero flag
- Behavioral simulation and waveform verification

## Tools Used
- Verilog HDL
- Xilinx Vivado

## Files
- alu.v → ALU design
- alu_tb.v → Testbench
- waveform.png → Simulation result
